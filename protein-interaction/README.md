# 🔬 Protein Interaction Network Analysis (TP53 Family)

## Objective
To compare protein interaction databases and analyze interaction patterns within TP53, TP63, and TP73 proteins.

---

## Databases Used
STRING, BioGRID, MINT, IntAct

---

## Work Performed
- Compared multiple databases based on interaction data and coverage  
- Collected interaction partners for TP53 family proteins  
- Analyzed overlap and uniqueness of interactions  
- Integrated data to construct a combined interaction network  
- Visualized network using Cytoscape  

---

## Key Analysis
- Identified common interacting proteins across TP53 family (e.g., MDM2, EP300)  
- Observed unique interactions specific to each protein  
- Detected highly connected nodes indicating hub proteins  
- Identified family-specific interaction modules
