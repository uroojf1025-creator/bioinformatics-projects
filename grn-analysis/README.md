# 🧬 Gene Regulatory Network (GRN) Construction and Comparison

## Objective
To construct gene regulatory networks using gene expression data and compare different inference methods.

---

## Data Used
- Synthetic gene expression dataset  
- ~100 genes and ~50 samples  

---

## Methods Applied
- ARACNE  
- Mutual Information (MI)  
- Relevance Network  

---

## Work Performed
- Generated gene expression dataset  
- Applied three different GRN inference methods  
- Constructed networks for each method  
- Compared network structures and connectivity patterns  

---

## Key Observations
- ARACNE network showed more specific interactions  
- MI network appeared denser with more connections  
- Relevance network included broader interactions  
- Some gene interactions were common across methods
